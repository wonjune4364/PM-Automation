NextCursor is a free Next.js 15 web app with 5 sequential AI documentation prompt generators (PRD → TRD → IA → Use Cases → Design), all producing Korean-language output. Monetized via Google AdSense. Optional OpenAI direct generation via user-supplied API key (BYOK, stored in localStorage).

**Why:** Owner is targeting the Korean vibe-coder and junior PM segment — an underserved market for Korean-language, structured technical documentation tooling.

**How to apply:** All PM research, feature recommendations, and market analyses should assume Korean-first output, free/AdSense monetization model, and the Cursor editor community as the primary distribution channel. Premium tier is a future option, not current reality.

Key positioning: "Pre-Cursor workflow" — users generate docs here, paste into Cursor as project context before coding. Closest structural competitor is Eraser.io (English-only). Largest indirect threat is Cursor itself adding native Korean doc generation.
