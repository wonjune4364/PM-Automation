## 역할 및 맥락

BJI-IT(NextCursor) 제품의 오너. PM Automation 프로젝트를 통해 5개 PM 에이전트(discovery, strategy, research, prd, lead)를 자동화하고 있다.

## 협업 스타일 관찰

- PRD 작성 요청 시 3개 분석 문서(discovery, strategy, research)를 입력으로 제공하는 패턴을 사용한다.
- 결과물은 한국어로 작성해야 한다 (PRD 명시 요건).
- 구체적이고 실행 가능한 내용을 선호한다 (모호한 일반론 지양).
- 8섹션 PRD 구조를 따른다 (Pawel Huryn 프레임워크 기반).

## 프로젝트 특이사항

- 제품명: BJI-IT / NextCursor
- 기술 스택: Next.js 15, App Router, shadcn-ui, Zustand, React Query
- 한국어 시장 우선 타겟
- Cursor 에디터 사용자 커뮤니티가 주요 배포 채널
