**North Star Metric:** Weekly Active Prompt Generators (unique users generating at least one prompt per week)

**Top assumptions to validate (ranked by risk):**
1. HIGH: Korean developers search for AI prompt generators in Korean for planning docs — validate via SEO keyword data
2. HIGH: Users willing to pay ₩9,900–₩19,900/month — validate with waitlist + pricing page before building paywall
3. HIGH: XML-structured prompts produce perceivably better output — validate with A/B test on output quality rating
4. MEDIUM: Sequential 5-generator workflow is used in one session — validate via cross-generator navigation analytics
5. MEDIUM: Vibe coders can complete forms without guidance — validate via usability tests

**UVP:** "The fastest way to get AI-ready, Korean planning documents — from idea to structured PRD, TRD, IA, Use Cases, and Design Guide in one sitting."

**Biggest strategic opportunity:** Make the sequential PRD→TRD→IA workflow explicit in the UX with a "continue to next step" flow to increase session depth and retention.
